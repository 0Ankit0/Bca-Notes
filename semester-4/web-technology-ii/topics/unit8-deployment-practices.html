Teaching outcomes

  • Deliver complete syllabus coverage with theory + practical integration.
  • Prepare students for assignments, term exam, lab exam, and viva.
  • Align lesson flow with the officially allocated 4 teaching hours.

8.1 Deployment mindset

Deployment turns classroom code into reliable production services with controlled configuration and security hardening.

8.2 Platform options

Discuss VPS, PaaS, and container platforms; compare simplicity, cost, and control.

8.3 Development vs production

Highlight debug mode risks, performance tuning needs, logging, and secret handling differences.

8.4 Production setup essentials

Teach environment variables, Gunicorn WSGI serving, reverse proxy patterns, and process supervision.

8.5 HTTPS with free certificates

Explain Let's Encrypt flow and certificate renewal automation.

8.6 Monitoring and security in deployment

Introduce metrics, logs, alerting, and incident response basics.

Classroom pacing for 4 hours

  1. Production architecture walkthrough (1h)
  2. Gunicorn + configuration exercise (1h)
  3. HTTPS setup demonstration (1h)
  4. Monitoring and deployment checklist (1h)
