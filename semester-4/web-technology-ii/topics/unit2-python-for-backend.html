Unit 2: Server-Side Programming with Python

Detailed teaching notes • 8 hours

Home / Semester 4 / Web Technology II / Unit 2 Detailed Notes

Teaching outcomes

  • Deliver complete syllabus coverage with theory + practical integration.
  • Prepare students for assignments, term exam, lab exam, and viva.
  • Align lesson flow with the officially allocated 8 teaching hours.

2.1 Python setup and environment discipline

Teach virtual environments, pip usage, and interpreter verification. Enforce reproducible setup practices from first class.

2.2 Core syntax and data handling

Cover primitive and container types, naming standards, casting, and expression evaluation.

2.3 Control structures and functions

Introduce if/elif/else, for/while loops, comprehensions, and modular function design with clean parameter/return contracts.

2.4 Data structures for backend tasks

Use practical examples: list for ordered records, dictionary for keyed student objects, tuple for immutable records, set for duplicate removal.

2.5 File handling + CSV/JSON

Demonstrate read/write/append modes, line-by-line processing, and proper context manager use. Convert CSV student data to JSON output as core lab.

2.6 OOP concepts

Build Student and Vehicle/Car classes with constructors, methods, and inheritance to connect OOP with future Flask model patterns.

Classroom pacing for 8 hours

  1. Syntax + control flow labs (2h)
  2. Data structures and algorithmic exercises (2h)
  3. File/CSV/JSON practical (2h)
  4. OOP implementation workshop (2h)
