4.1 Database foundations for web apps

Discuss persistence needs, entity relationships, and selecting relational or NoSQL systems based on data consistency and query patterns.

4.2 Relational vs NoSQL

Relational DBs enforce schema and ACID properties; NoSQL databases offer flexible models for high-scale or document-centric use-cases.

4.3 Flask database connectivity

Use SQLAlchemy for ORM mapping, connection lifecycle management, and abstraction from raw SQL in beginner projects.

4.4 CRUD operations through templates

Implement create/read/update/delete flows with forms and dynamic tables for a students dataset.

4.5 Transactions and reliability

Explain commit/rollback semantics and demonstrate error-safe transaction handling during updates.

Classroom pacing for 8 hours

  1. Schema design and DB setup (2h)
  2. Read and insert operations (2h)
  3. Update/delete operations (2h)
  4. Transaction handling + integration review (2h)
