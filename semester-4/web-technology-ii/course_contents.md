## Practical works coverage map

- HTTP request/response simulation -> Unit 1
- Web server setup (Apache/Nginx) -> Unit 1 and Unit 8
- Python fundamentals, file handling, CSV/JSON, OOP -> Unit 2
- Flask setup, forms, templates, session, validation, file upload, email -> Unit 3
- MySQL + SQLAlchemy integration, dynamic pages, CRUD -> Unit 4
- API development and consuming APIs with `requests` -> Unit 6
- Security controls for login/session/password hashing -> Unit 7
- Deployment with Gunicorn and HTTPS certificates -> Unit 8
